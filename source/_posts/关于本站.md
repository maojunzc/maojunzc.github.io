## 博客简介

本站是基于 **Hexo** 框架构建的个人博客，托管于 **GitHub Pages**，使用 [Ayer](https://github.com/shen-yu/hexo-theme-ayer) 主题并深度定制了暗色炫酷风格。

### 功能特性

- 🎨 **暗色炫酷 UI** — 毛玻璃侧边栏、渐变文字、粒子动画背景
- 🌊 **3D 水波特效** — 首页交互式波浪网格，鼠标移动改变视角
- 💬 **Waline 评论系统** — 自部署（Vercel + Neon PostgreSQL），支持邮件通知和 GitHub 登录管理
- 🔍 **本地搜索** — 支持文章内容检索，快捷键 `/` 快速聚焦
- 📱 **响应式布局** — 适配桌面端和移动端
- 🖼️ **PhotoSwipe** — 图片点击放大浏览
- 🎵 **APlayer 音乐** — 支持音乐播放
- 🖥️ **Blog Manager 桌面工具** — 一键发布文章、图片上传（多图床）、Git 自动管理

### 前端技术栈

| 技术 | 用途 |
|------|------|
| [Hexo](https://hexo.io/) | 博客框架 v5.4.2 |

## Blog Manager 桌面工具

`Blog Manager` 是一个使用 **Python + Tkinter** 构建的桌面端博客管理工具，支持 ttkbootstrap 主题，可以让你在不接触命令行的情况下管理博客。

### 功能一览

| 功能 | 说明 |
|------|------|
| 📝 文章管理 | 新建、编辑、导入、删除 Markdown 文章 |
| 🚀 一键发布 | 自动处理 front-matter、图片上传、Git 提交推送 |
| 🖼️ 多图床 | 本地 / GitHub / ImgBB / SM.MS 四种模式 |
| 📥 拖拽导入 | 拖拽 .md 文件和图片到窗口即可处理 |
| ⚙️ 完整设置 | Git 配置、图床密钥、写作模式自由切换 |
| 🌐 本地预览 | 一键启动 Hexo server 预览 |
| 🎨 写作模式 | 经典灰色白底黑字模式，专注写作 |

### 下载方式

Blog Manager 为 Windows 单文件 exe，下载即可使用，无需安装 Python 环境。

**下载链接：**
> [https://github.com/maojunzc/maojunzc.github.io/raw/main/blog-manager/dist/BlogManager.exe](https://github.com/maojunzc/maojunzc.github.io/raw/main/blog-manager/dist/BlogManager.exe)

下载后双击运行，首次使用请在设置中配置博客仓库路径。

### 使用说明

1. 下载 `BlogManager.exe` 并运行
2. 点击 ⚙️ 设置 → 填写 **仓库路径**（即你克隆博客的本地目录）
3. 配置 Git 用户名和邮箱（用于自动提交）
4. 在 **发布** 页面选择 .md 文件，点击 **一键发布**
5. 支持拖拽 .md 文件或图片到窗口快速处理

> ⚠️ 需要本地已安装 Git 并配置好 SSH/HTTPS 连接 GitHub。
